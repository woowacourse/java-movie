# 영화 예매

## 기능 목록

* 상영 영화 목록 보여주기
* 영화 선택하기
    * 선택한 영화 정보 출력하기
    * 상영 영화 목록에 없는 번호 입력시 다시 입력받기
    * 상영 시간이 지난 영화 다시 입력받기
* 선택한 영화의 시간 선택하기
    * 해당 시간이 없는 경우 다시 입력 받기
* 예약 인원 입력하기
    * 예약 가능 인원이 아닐 경우 다시 입력받기
* 예약 내역 출력하기
* 사용할 포인트 입력하기
    * 0 이상의 수 만 입력 가능
* 결제 수단 선택하기
    * 1, 2 이외의 숫자를 입력받으면 다시 입력받기
* 결제 방식에 따른 결제 금액 할인하기
