movie 기능 목록

 - 입력 받은 예약 영화를 출력한다.

 - 예약할 시간표를 입력받는다.

 - 입력받은 시간표를 저장한다.

 - 입력받은 시간표가 숫자가 아닐경우 를 예외처리 한다.

 - 입력받은 시간표가 범위를 벗어날 경우를 예외처리 한다.

 - 예약할 인원을 입력받는다.

 - 예약 인원의 숫자를 확인하고 숫자가 아닐경우를 예외처리 한다.

 - 예약 가능 인원보다 예약 인원이 더 많을경우 메세지 출력.

 - 예약 인원이 0이하 일경우 메세지 출력.

 - 예약 종료후 결제 진행을 입력받는다.

 - 결제 진행 값이 숫자가 아닐경우 예외처리

 - 결제 진행 입력 값이 1 또는 2가 아닐 경우 예외 처리 한다.

 - 포인트 사용금액을 입력받는다.

 - 포인트가 음의 값을 가질경우 예외 처리 한다.

 - 포인트가 숫자가 아닐경우 예외 처리 한다.

 - 신용카드 , 현금 입력을 받는다.

 - 신용카드, 현금의 숫자가 아닐경우, 포인트 범위를 벗어날경우 예외 처리 한다.

 - 최종 결제 금액을 출력한다.
